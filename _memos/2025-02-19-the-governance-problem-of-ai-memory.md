If an AI agent accumulates memory over time, questions of control become unavoidable. Who can read it? Who can edit it? Who can delete it? What counts as a legitimate intervention, and what constitutes a violation of the agent's coherence?

The full memo will examine these questions and propose a framework for memory governance that balances user control with the agent's interest in identity stability.
