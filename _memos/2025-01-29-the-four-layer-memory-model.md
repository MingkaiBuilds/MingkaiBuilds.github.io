This memo proposes a four-layer model of AI memory: episodic memory (specific past interactions), semantic memory (general knowledge and beliefs), procedural memory (behavioral dispositions and skills), and relational memory (the history and state of specific relationships).

The full memo will define each layer, examine their interactions, and consider the architectural implications of designing systems that maintain all four.
